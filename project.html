Comanager App – UI/UX Case Study

Project Overview

Role: UI/UX Designer

Tools: Figma, Adobe XD

Duration: 4 weeks

Goal: To design an user-friendly interface for automobile detailers, garage owners.

Problem Statement

Users found it difficult to handle their garage detailing studio manually so comanager will help them to manage there business.

Design Process

  • User Research: Conducted surveys and interviews to understand user needs.
  • Wireframing: Created low-fidelity wireframes to map out user flows.
  • Prototyping: Developed interactive prototypes using Figma.
  • Usability Testing: Performed tests with target users and iterated based on feedback.

Key Features

  • Booking & Scheduling: Manage customer appointments and service schedules seamlessly.
  • Vendor Management: Track and coordinate with suppliers for parts and services.
  • Inventory Control: Monitor stock levels, manage parts inventory, and receive alerts for low stock.
  • Accounts & Invoicing: Handle billing, generate invoices, and manage financial transactions.

Visual Design

The visual design focuses on a minimalist aesthetic with a Blue color scheme to enhance user immersion. Typography and iconography were chosen for clarity and modern appeal.

Conclusion

The redesign resulted in a 25% increase in user engagement and a 15% improvement in user satisfaction scores. The new design successfully addressed the initial pain points and enhanced the overall user experience.
